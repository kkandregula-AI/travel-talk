How to Use
Travel Talk AI
1
Enter your API Key 🔑
Tap the 🔑 icon in the header. Paste your Anthropic API key (starts with sk-ant-). It's stored locally — never sent anywhere except Anthropic.
2
Set Your Travel Profile 🗂️
Go to the Profile tab. Enter your city, hotel, dietary needs (vegetarian, halal…), and communication style. This makes every translation context-aware — not literal.
3
Translate a Phrase 🌐
On the Translate tab, type or tap 🎤 to speak your phrase. Choose source & target language, then tap Translate →. The AI uses your travel profile for culturally accurate output.
4
Speak, Copy & Save 🔊
After translating, tap 🔊 Speak to hear it aloud in the target language. 📋 Copy to share it. 💾 Save adds it to your phrase memory for future trips.
5
Phrase Memory — RAG Recall 🧠
As you type, past saved phrases appear as 💡 Recall cards. Tap one to instantly reuse a translation. The Memory tab lists all saved phrases — searchable by keyword, language, or tag.
6
Cache Offline Phrases 📥
Tap 📥 in the header to bulk-translate 10 essential travel phrases (doctor, bill, allergies…) into your target language and save them offline — no internet needed later.
💡
Pro Tip: Add to Home Screen on iOS (Share → Add to Home Screen) or Android (browser menu → Install App) to use it as a full PWA with offline memory access.
